HTML: La Estructura

HTML (HyperText Markup Language) es el esqueleto de cualquier página web. Define el significado y la estructura del contenido web.

Etiquetas Semánticas Importantes

HTML5 introdujo etiquetas que explican qué es el contenido, no solo cómo se ve:

  • <header>: Para cabeceras y logotipos.
  • <nav>: Para menús de navegación.
  • <section>: Para agrupar contenido temático.
  • <article>: Para contenido independiente (como un blog).
